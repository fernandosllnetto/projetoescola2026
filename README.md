## 📌 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de praticar e evoluir habilidades em **Java**, aplicando conceitos fundamentais de programação e organização de código.

A ideia é evoluir gradualmente este repositório para um sistema mais completo, podendo futuramente incluir API, banco de dados e interface.
